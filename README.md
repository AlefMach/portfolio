# Portfolio - Alef Machado

Portfolio pessoal para apresentar projetos, experiencia e forma de trabalho em desenvolvimento Full Stack, IA aplicada, automacao, observabilidade e integracoes.

## Stack

- React
- TypeScript
- Vite
- Material UI
- Framer Motion

## Comandos

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Estrutura

- `src/components/sections`: secoes principais da pagina inicial.
- `src/components/common`: componentes compartilhados, como toggles e links sociais.
- `src/i18n/locales`: textos em portugues e ingles.
- `public/images`: imagens usadas no perfil e nos projetos.

## Deploy

O projeto esta configurado para GitHub Pages com base `/portfolio/`.

```bash
npm run deploy
```
